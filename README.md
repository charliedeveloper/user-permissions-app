# User Permissions App

This project contains:

- `frontend/` - Angular application
- `api/` - ASP.NET Core Web API

## Features

- Create groups
- Create permissions
- Assign permissions to users

## Project Structure

```text
user-permissions-app/
  frontend/
  api/
```

## How to Run

### Frontend

```bash
cd frontend
npm install
ng serve
```

### API

```bash
cd api
dotnet restore
dotnet run
```
